
# Manual da Jammer

Este repositório contém guias rápidos para ajudar durante Game Jams.  
Ele funciona como uma enciclopédia prática com instruções, boas práticas e dicas para quem quer se organizar durante a jam.

## Como colaborar

1. Faça um **fork** deste repositório  
2. Clone o seu fork no computador:

```bash
git clone https://github.com/SEU_USUARIO/manual-jammer.git
````

3. Crie uma branch para a sua contribuição:

```bash
git checkout -b minha-contribuicao
```

4. Faça suas alterações nos arquivos dentro da pasta `guias/`
5. Commite suas alterações:

```bash
git add .
git commit -m "Descrição do que você alterou"
```

6. Faça push para o seu fork:

```bash
git push origin minha-contribuicao
```

7. Abra um **Pull Request** neste repositório.

---

## Regras de contribuição

* Mantenha os guias organizados dentro da pasta `guias/`
* Não altere arquivos de outros guias sem necessidade
* Use nomes de arquivos claros, sem espaços (ex: `guia-jammer-github-game-jam.md`)
* Atualize apenas o conteúdo dos guias, caso crie um guia: lembre de atualizar a lista de guias disponíveis no `README.md`

---

## Objetivo

Ter um **manual de referência rápida** para jammers iniciantes ou experientes, que funcione durante a jam e ajude a evitar erros comuns com Git, GitHub, Unity, Godot e outras ferramentas.
